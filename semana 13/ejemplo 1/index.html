PHP

PHP es un lenguaje de programación de uso general que se adapta especialmente al desarrollo web.

Java

Java es un lenguaje de programación orientado a objetos ampliamente utilizado en el desarrollo de aplicaciones empresariales y de escritorio.

C#

C# es un lenguaje de programación desarrollado por Microsoft y se utiliza principalmente en el desarrollo de aplicaciones de escritorio y juegos para Windows.
